## `pierre_export_locales_json`

**Action** : `wp_ajax_pierre_export_locales_json`

**Nonce** : `pierre_admin_ajax`

**Capability** : `pierre_view_dashboard`

**Paramètres** : Aucun

**Réponse** : Fichier JSON téléchargeable
