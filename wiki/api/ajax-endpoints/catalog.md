# Endpoints Catalogue de Projets

Le catalogue de projets contient de nombreux endpoints pour gérer le catalogue. Consultez le code source pour les détails complets de tous les endpoints.

## Endpoints Principaux

- `pierre_admin_rebuild_catalog` : Reconstruit le catalogue
- `pierre_admin_fetch_catalog` : Récupère une page du catalogue (avec pagination et filtres)
- `pierre_admin_save_catalog_settings` : Sauvegarde les réglages du catalogue
- `pierre_admin_get_catalog_status` : Récupère le statut du catalogue
- `pierre_admin_schedule_catalog` : Planifie une reconstruction
- `pierre_admin_inspect_catalog` : Inspecte le contenu
- `pierre_admin_get_catalog_stats` : Récupère les statistiques
- `pierre_admin_get_catalog_progress` : Récupère la progression
- `pierre_admin_get_catalog_errors` : Récupère les erreurs
- `pierre_admin_render_catalog_browser` : Génère le markup HTML
- `pierre_admin_purge_catalog` : Purge le catalogue
- `pierre_admin_reset_catalog` : Réinitialise le catalogue
- `pierre_admin_catalog_export_to_library` : Exporte vers la bibliothèque
- `pierre_admin_catalog_import_from_library` : Importe depuis la bibliothèque
- `pierre_admin_export_catalog_json` : Exporte en JSON
- `pierre_admin_export_catalog_csv` : Exporte en CSV
- `pierre_admin_export_catalog_errors_json` : Exporte les erreurs en JSON
- `pierre_admin_export_catalog_errors_csv` : Exporte les erreurs en CSV

Pour les détails complets de chaque endpoint, consultez le fichier original ou le code source.
